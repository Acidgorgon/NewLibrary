The Boot Sequence

Every professional script starts with a great loader. NextLevel's InitLoad is fully customizable and supports a hidden "Silent Mode".

Step 1: The Premium Loader

Library:InitLoad(options: table)
Library:InitLoad({
    Title = "NextLevel",
    SubTitle = "v0.06 Premium",
    Icon = 89738140174311,          -- asset id
    Duration = 3,                   -- seconds to stay visible
    AccentColor = Color3.fromRGB(255, 255, 255),
    Statuses = {
        "Checking license...",
        "Fetching data...",
        "Injecting..."
    },
    Silent = false -- Set to true for stealth execution
})

Step 2: Silent Execution

If Silent is true (or if the user enabled "Silent Execution" in their saved config), the loader will be completely skipped and the UI will start hidden.

How to use Webhooks

Webhooks are used to send data (logs, notifications, or player data) from Roblox directly to a Discord channel.

  1. Create a Webhook: In Discord, go to Server Settings > Integrations > Webhooks > New Webhook.
  2. Copy URL: Copy the Webhook URL and paste it into the UI's Webhook widget.
  3. Send Data: Use the following snippet to send logs manually:
local function SendLog(msg)
    local url = Window.Flags["WebhookFlag"] -- The flag you used in Section:Webhook
    if url and url ~= "" then
        request({
            Url = url,
            Method = "POST",
            Headers = { ["Content-Type"] = "application/json" },
            Body = game:GetService("HttpService"):JSONEncode({
                content = msg,
                username = "NextLevel Logs"
            })
        })
    end
end

-- Example usage:
SendLog("Player " .. game.Players.LocalPlayer.Name .. " has injected the script!")

Window Utilities

Advanced methods for managing connections, tooltips, and window state.

Window:Connect(signal, callback)

Connects a signal and automatically disconnects it when the UI is unloaded.

Window:OnUnload(callback)

Registers a function to be executed when the library is unloaded (via Panic Key or Unload button). Perfect for stopping loops or resetting player properties.

Window:AddTooltip(element, text)

Adds a hover-to-view description to any UI element.

Window:SetPanicKey(KeyCode)

Set a key that instantly unloads the library when pressed.

Window:IgnoreFlag(Flag)

Excludes a specific flag from being saved in configurations.

Window:ListConfigs()

Returns an array of strings containing all saved configuration names.
